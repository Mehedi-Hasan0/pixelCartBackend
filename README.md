## API Reference

#### Create user

```http
  POST /api/v1/users/create-user
```

#### Create seller

```http
  POST /api/v1/sellers/create-seller
```

#### Details

| Parameter | Type   | Required field                     |
| :-------- | :----- | :--------------------------------- |
| `null`    | `POST` | Name, email, password, phoneNumber |
